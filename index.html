Publishing a static site with GitHub Pages is easy:

  1. Go to your GitHub account (ie: github.com/yourusername -- that's mine)
  2. Make a repo called yourusername.github.io (ie: yourusername.github.io -- in my case)
  3. Commit a file named index.html -- that will be your homepage
  4. Now visit the site yourusername.github.io in your browser. It may take a minute to show up, but that's it!
